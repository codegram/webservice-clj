# webservice-clj

An example webservice written in Clojure for the talk [Building a Clojure
webservice in 10 minutes][slides].

It is a small Clojure webservice backed by MongoDB.

## Usage

To start the server:

    $ lein run

To run the tests:

    $ lein test
